Short description

This interactive webpage showcases the legendary figures of East Coast and West Coast hip-hop. Using tabs, it allows users to explore influential artists like Diddy, Biggie, and Jay-Z from the East Coast, and Ice Cube, Tupac, and Snoop Dogg from the West Coast. The design features smooth animations, a stylish dark theme with a starry background, and includes YouTube links for music highlights. The layout is clean and responsive, offering a modern portfolio-style presentation of hip-hop culture.

Technologies Used

1. HTML (HyperText Markup Language) Purpose: The backbone of the webpage; it structures all the content like headings, buttons, images, and text. Example in your code: div, h1, img, button, etc.
2. CSS (Cascading Style Sheets) Purpose: Controls the styling and layout of the webpage, such as colors, fonts, spacing, and animations. Used for: Gradient backgrounds Card styles and hover effects Responsive grid layout Tab transitions and fade animations
3. JavaScript Purpose: Adds interactivity to the site, such as switching between East Coast and West Coast tabs. Used for: Tab button functionality Dynamically showing and hiding content
4. Embedded Media (YouTube Links) Purpose: Direct links to songs that give users an immersive experience. Used for: Letting users watch music videos directly from the artist cards.
5. External Image Hosting Purpose: Hosts artist images using URLs from the internet. Used for: Displaying artist portraits in the cards without storing images locally.
